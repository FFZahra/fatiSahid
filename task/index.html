Available API Services
API Name API Description
1. Country Neighbours

Accepts a country name and returns a list of countries in its neighbourhood.
2. Street Names

Accepts a country name and an optional postal code, and returns a list of street names that start with the characters typed in the search box.

Results are currently available only for Australia, Iceland, Norway, Singapore & Spain.

3. Oceans

Accepts a longitude and latitude value, and returns the name of any ocean(s) in the given location.
