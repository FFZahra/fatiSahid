Available API Services
API Name API Description
1. Nearby Postal Codes

Accepts a country name, a postal code, and a radius and returns a list of postal codes and their place names, within the given radius.
2. Location Lookup

Accepts an address(or part of an address) and an optional country name, and returns the latitude and longitude data for the given location.

This service is currently available for the listed countries only.
3. Address Lookup

Accepts a longitude and latitude value, and returns the nearest address to the given location.

This service is currently available for the listed countries only.
